# promotion_engine

This service calculates the total amount of the cart.
The rules and product prices defined are:

Unit price for SKU IDs 
A 50
B 30
C 20
D 15

Active Promotions

3 of A's for 130
2 of B's for 45 
C & D for 30 


To run the program, run main.go by using the following command:

go run main.go
